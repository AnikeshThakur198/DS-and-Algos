Some Imp Methods in order to find subsets of a String:

1.) String.substring(int beginIndex):
	This method of String class is used to print or return a part of a string

	eg: 	String str = "Hello";
		System.out.println(str.substring(1));		
	output:
		"ello"

2.) String.substring(int beginIndex, int endIndex);
	This method  of String class is also used to print a part of a string but it has  
	little more features.

	eg:	String str = "Hello";
		System.out.println(str.substring(1, 4));
	output:
		"ell"
